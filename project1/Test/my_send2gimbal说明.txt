### 方向判断
- deta_x ：
  - 正值 ：目标在基准点的 右侧
  - 负值 ：目标在基准点的 左侧
  - 0 ：目标在基准点的水平中心线上
- deta_y ：
  - 正值 ：目标在基准点的 下方
  - 负值 ：目标在基准点的 上方
  - 0 ：目标在基准点的垂直中心线上

### 偏差程度
- 绝对值越大，偏差越大
- 绝对值越小，目标越接近基准点
- 当 deta_x 和 deta_y 都接近 0 时，目标位于激光点上


(0,0) ─────────→ x+
  │
  │
  ↓
  y+


✅ dx（左右控制）
dx = center_x - base_x
情况	含义	dx 符号
目标在右边	center_x > base_x	dx > 0
目标在左边	center_x < base_x	dx < 0
👉 所以：
dx > 0 → 云台往右转
dx < 0 → 云台往左转

✅ dy（上下控制）
dy = base_y - center_y
注意这里反了一次
情况	含义	dy 符号
目标在上面	center_y < base_y	dy > 0
目标在下面	center_y > base_y	dy < 0
👉 所以：
dy > 0 → 云台往上抬
dy < 0 → 云台往下压
