# 激光视差补偿原理文档

## 1. 问题背景

激光器和摄像头安装在小车上，但两者存在物理安装间距。由于这个间距，摄像头"看到"的激光落点位置会随目标距离变化而偏移——这就是**视差效应**。

```
摄像头
  |
  |  ← 垂直安装间距 d
  |
激光器  ──────────────→  靶板
              距离 D
```

距离越近，视差角越大，激光点在图像中的偏移越明显；距离越远，视差角越小，偏移越小。

因此需要根据当前距离动态修正激光点在图像中的预测坐标，使程序能准确计算激光落点与靶心的误差。

---

## 2. 补偿链路总览

```
摄像头拍到靶板
      ↓
检测靶板轮廓，计算像素面积 A
      ↓  【公式一】
估算摄像头到靶板的距离 D（mm）
      ↓  【公式二】
限幅，防止越界
      ↓  【公式三】
线性插值，得到激光点 Y 坐标（px）

X 坐标固定 = 322 px（不参与计算）
```

---

## 3. 三个核心公式

### 公式一：像素面积 → 距离

**物理原理**：物体在图像中的投影面积与距离的平方成反比（`S ∝ 1/D²`）。

**数学公式**：

```
D = D_ref × √(A_ref / A)
```

**代码位置**：`estimate_distance_by_area()` 函数，第 390 行

```python
distance_mm = REFERENCE_DISTANCE_MM * np.sqrt(REFERENCE_AREA_PIXELS / area_pixels)
```

**参数说明**：

| 参数 | 变量名 | 值 | 单位 |
|------|--------|----|------|
| 参考距离 | `REFERENCE_DISTANCE_MM` | 1500 | mm |
| 参考面积 | `REFERENCE_AREA_PIXELS` | 8412 | px² |
| 当前面积 | `area_pixels` | 实时检测 | px² |
| 输出距离 | `distance_mm` | 计算结果 | mm |

> 参考值的标定方法：将 A4 纸靶板放在距摄像头 1500mm 处，记录程序检测到的像素面积，填入 `REFERENCE_AREA_PIXELS`。

---

### 公式二：距离限幅

防止超出有效范围的距离值导致坐标异常。

**数学公式**：

```
D_clamped = clamp(D, 500, 1500)
```

**代码位置**：`calculate_laser_position_by_distance()` 函数，第 494 行

```python
distance_clamped = max(MIN_DISTANCE_MM, min(MAX_DISTANCE_MM, distance_mm))
```

**参数说明**：

| 参数 | 变量名 | 值 | 单位 |
|------|--------|----|------|
| 最近有效距离 | `MIN_DISTANCE_MM` | 500 | mm |
| 最远有效距离 | `MAX_DISTANCE_MM` | 1500 | mm |

---

### 公式三：距离 → 激光点 Y 坐标（线性插值）

**数学公式**：

```
Y = Y1 + (D_clamped - D1) × (Y2 - Y1) / (D2 - D1)
```

**代码位置**：`calculate_laser_position_by_distance()` 函数，第 497 行

```python
y = LASER_Y_MIN_DISTANCE + (distance_clamped - MIN_DISTANCE_MM) * \
    (LASER_Y_MAX_DISTANCE - LASER_Y_MIN_DISTANCE) / (MAX_DISTANCE_MM - MIN_DISTANCE_MM)
```

**参数说明**：

| 参数 | 变量名 | 值 | 单位 |
|------|--------|----|------|
| 最近距离对应 Y | `LASER_Y_MIN_DISTANCE` | 240 | px |
| 最远距离对应 Y | `LASER_Y_MAX_DISTANCE` | 250 | px |
| 最近标定距离 D1 | `MIN_DISTANCE_MM` | 500 | mm |
| 最远标定距离 D2 | `MAX_DISTANCE_MM` | 1500 | mm |

**映射示例**：

| 距离 D | 计算过程 | Y 坐标 |
|--------|----------|--------|
| 500 mm | 240 + 0 | 240 px |
| 750 mm | 240 + 2.5 | 242 px |
| 1000 mm | 240 + 5 | 245 px |
| 1250 mm | 240 + 7.5 | 247 px |
| 1500 mm | 240 + 10 | 250 px |

**X 坐标**：固定为 `LASER_X_FIXED = 322 px`，由激光器水平安装位置决定，不随距离变化。

---

## 4. 标定方法

### 标定公式一（面积→距离）

1. 将 A4 纸靶板垂直放置在距摄像头 **1500mm** 处
2. 运行程序，读取调试输出中的检测像素面积
3. 将该面积填入 `REFERENCE_AREA_PIXELS`

### 标定公式三（距离→Y坐标）

1. 将小车放在距靶板 **500mm** 处，打开激光
2. 在摄像头画面中找到激光实际落点的 **Y 像素坐标**，填入 `LASER_Y_MIN_DISTANCE`
3. 将小车移到 **1500mm** 处，同样找激光实际落点的 **Y 像素坐标**，填入 `LASER_Y_MAX_DISTANCE`

> 注意：测量的是**激光实际落点**的 Y 坐标，不是靶心中心的 Y 坐标。

---

## 5. 精度说明

当前方案使用**两点线性插值**近似真实视差曲线。真实视差是反比关系（`offset = f × d / D`，双曲线），线性近似在中间段存在系统误差，但由于 Y 总变化量仅 10px，工程上可以接受。

若需提高精度，可在多个距离点（如 500 / 750 / 1000 / 1250 / 1500mm）各测一次激光落点 Y 坐标，改用分段线性插值：

```python
import numpy as np

distances = [500, 750, 1000, 1250, 1500]  # 实测距离（mm）
y_values  = [240, 243, 246,  248,  250]   # 对应激光落点Y坐标（px）

y = int(np.interp(estimated_distance, distances, y_values))
```
